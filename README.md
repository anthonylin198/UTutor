## About The Project

UTutor is an open source marketplace application for University tutors to buy and sell tutoring services. Our goal is to provide affordable and effective tutoring at the university level.

### Built With

- React
- Redux
- Node.js
- MongoDB

### Installation

1. Clone the repo

```sh
git clone https://github.com/anthonylin198/UTutor.git
```

2. Install NPM packages

```sh
npm install
```

3. Start the Project

```sh
npm start
```
